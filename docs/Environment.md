# Atom
Find in-file items with https://atom.io/packages/todo-show
```
Find Todos: FIXME, TODO, CHANGED, XXX, IDEA, HACK, NOTE, REVIEW
Find Rejex: /\b(${TODOS})[:;.,]?\d*($|\s.*$|\(.*$)/g
Ignore Paths: .meteor, node_modules
```
